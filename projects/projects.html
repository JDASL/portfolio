🌍 World Clock App

This was my second coding project, where I built a real-time World Clock usingHTML, CSS, JavaScript and the Moment Timezone API. I wanted to challenge myself by creating something dynamic and functional.

The app displays accurate, real-time clocks for different global cities such as Oslo 🇳🇴, Bangkok 🇹🇭, and New York 🇺🇸. Users can also select a city from a dropdown menu and view the local time instantly.

This project helped me learn how to:

  • Use JavaScript's setInterval() to update live clocks
  • Integrate Moment.js and timezone libraries
  • Build responsive layouts and interactive UI with advanced CSS
